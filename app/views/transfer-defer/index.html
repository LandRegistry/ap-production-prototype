Transfer - deferral for discharges - IC6 PROM-122

When an application does not include evidence of discharge, and we need to request it because there are restricitions in the register, a caseworker:

  • will know evidence is required
  • will write a requisition letter (C90), if this is the only reason to requisition
  • use legacy system, PC Corres to write the letter

the system will:

  • add a PAS note, allowing the aplication to be stored until the evidence is received
  • put the application in a new 'pot' (category of work) to be picked up for evidence to be checked
  • run all the business rules again, to see if anything else has changed

A different caseworker (probably):

  • will pick up the application
  • check the evidence (electronic discharge via the daylist)
  • do any additinal tasks required - if circumstances have changed (Business rules will establish this)
  • complete the applciation - markoff, or send it to legacy processing
