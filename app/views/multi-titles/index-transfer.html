Multi titles

This feature enables a caseworker to process DRS single transfer multi titles (up to 25 titles) in Application Processing. The linked demo represents the first state version in which apportionment will be handled in CORA, with a PAS notes added in AP to prevent mark off.

This demo includes:

  • An application with 3 titles (address, land to the back of and associated garage) submitted for private individual borrowers.
  • Routing for different transfer scenarios (single addressable property, range of properties, portfolio of unrelated properties).

This feature will be iterated to enable apportionment in later phases.

When using this demo multiple times, the data submitted may need to be cleared for the correct pages to show. This can be acheived by using the 'Clear data' link in the footer of the page. Clear the data and then use the browser back button to return to your place in the flow.
