Check your answers

This feature enables a caseworker to check and change the answers they have provided in Application Processing. The linked demo represents the MVP build, which will enable users to view all provided answers and change them by returning to the transaction and its associated tasks.

This demo includes:

  • an application involving a combination of transactions (Discharge, Transfer for value, Charge, RX1) submitted by two private individuals.
  • a scenario which begins at the end of processing, allowing the user to view the 'check your answers' tab as part of the final summary page.

The demo's path assumes that the application has been processed. The user notes that the proprietors names must be 'Scotts' instead of 'Scott' as was previously entered. You can change this response by following the change link next to the relevant task and returning to the transaction.

Note: Changing the names of the transferees is the only interaction currently programmed into this demo. Other links will not function here but will be accessible in the live release.
