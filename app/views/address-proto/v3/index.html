What are the steps that the user is asked to follow in the prototype?

  1. Users starts from the transferees page. The transferee names are pre-populated by the system.
  2. They then get into a page showing an empty address table. They are so required to start populating the table by pressing on 'add address'.
  3. When adding address number 1, they have to select 'UK postal address', and then fill the details accordingly before pressing 'save address'.
  4. Address number 1 will so get added to the table.
  5. Users now have to enter address number 2. They do so by pressing on 'add address'.
  6. Users will then have to select 'email address' when asked about the type of address they want to add for address number 2, and fill the details in.
  7. Saving address number 2 will then add the email address to the table.
  8. At this point the number of addresses that can be added in the prototype is reached , and the user can press next to move to the next page of the Transfer flow, having answered the 'Other than UK postal address and email address are there any more addresses for service that need to be added?' question.
